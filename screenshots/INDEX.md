## 🚀 How to Generate Screenshots

1. Run the application:
   ```bash
   python app.py
   ```

2. Open in browser:
   ```
   http://127.0.0.1:5000
   ```

3. Login with demo credentials:
   ```
   Username: sachin
   Password: password1
   ```

4. Navigate through different sections and capture screenshots using:
   - Chrome: Ctrl + Shift + S
   - DevTools: F12 → Ctrl + Shift + P → Screenshot
   - Third-party tools: Snagit, Greenshot, ShareX
